⌚ ClaudeApprove

Approve Claude Code permission prompts from your Apple Watch. When Claude needs your OK to run a command, your wrist buzzes with real Approve/Deny buttons. A tap unblocks Claude instantly — from the next room or the next city.

Get started

  1. Get the ClaudeApprove app for iPhone + Apple Watch (App Store link coming soon — currently in beta).
  2. Open it and tap Set Up — it shows a pairing code.
  3. On the Mac where you run Claude Code: 
    curl -fsSL https://christopherrathbun.com/claude-approve/install.sh | bash
    and enter the code.
  4. Run ~/.claude/watch-approve/away on when you step away. That's it.

How it works

A Claude Code hook intercepts permission prompts and sends them to this site's backend, which pushes a notification to your devices. Your tap resolves the request in about 100 ms via long-polling. If you don't respond within a few minutes, Claude falls back to the normal terminal prompt — the hook fails safe in every error case.

Privacy & security

  • Pairing uses a one-time code; your account is an anonymous random token — no email, no sign-up.
  • Approval requests (tool name, a one-line summary, working directory) are stored only until resolved and pruned within minutes.
  • An approval grants that single tool call — nothing is allowlisted.
